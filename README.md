# 🎨 Magic Background Generator

Uma aplicação web desenvolvida com HTML, CSS e JavaScript para criação e visualização de backgrounds animados.

## 🚀 Demonstração

🔗 **Projeto Online:** (https://raynner-victor.github.io/fundo-magico/)

## 📌 Sobre o Projeto

O objetivo deste projeto foi desenvolver uma interface capaz de gerar e exibir backgrounds animados de forma simples e intuitiva, explorando conceitos fundamentais do desenvolvimento front-end.

Durante a construção da aplicação, foram aplicados conhecimentos de estruturação semântica, estilização avançada, responsividade e manipulação do DOM para criar uma experiência visual dinâmica e agradável.

## 🎯 Desafio

Criar uma solução capaz de exibir efeitos visuais animados em tempo real mantendo uma interface simples, organizada e responsiva.

## 💡 Solução

Foi desenvolvida uma aplicação utilizando HTML, CSS e JavaScript que permite visualizar backgrounds animados diretamente no navegador, com foco em organização de código, performance e experiência do usuário.

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (ES6)

## 📚 Principais Conceitos Praticados

* Estruturação semântica com HTML.
* Layouts responsivos.
* Manipulação do DOM.
* Organização de código.
* Animações e efeitos visuais com CSS.
* Boas práticas de desenvolvimento Front-End.
